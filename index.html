Background-images,Let's Go!


Properties


There are a few diffrent background Properties:

background-image

     This Property is the location of the image itself, for example:
     h1 { backgroung-image: url('http://website.com/img/background.jpg'); }

     or

     h1 { backgroung-image: url('images/background.jpg'); }

background-color

     This allows as to add color to a background.An example in CSS would like this:
     h1 { backgroung-color:green; }

background-repeat

     This is how the image will repeat.The value can be any of the following

      background-repeat: repeat;

      This repaets on both the X and Y axis, and will essentially tile the image across the
       entire background.


      background-repeat: repeat-x;
      This repeats the background on the X axis, or horizontally.

      background-repeat: repeat-y;
      This repeats the background on the Y axis, or vertically.

      background-repeat: no-repeat;

      This show the image only once, without repeating.

background-position

     This is the actual position of the background image within it's container.The valuer can be:
     background-position: top;
     background-position: right;
     background-position: bottom;
     background-position: left;
     background-position: center;

     It can also be a percentage, length, or combination,like so:
     background-position: 50px 20%;

Shorthand

            
  There's a quick way around background images, and you don't necessarily have to type out all of
  the above code to set up your background imgaes the way you want them.

background

     This "background" shorthand can handle them all in one!
     background: green url('images/background.jpg') top left no-repeat;
